# Repo Notes

1. This repo is a small Astro static site used as a LeadTruffle demo lab on GitHub Pages, where we showcase widgets, autoresponder demos, and candidate pull request work.
2. After making changes, always run the project checks and validate they pass before finishing. Use `npm run check`.
3. LeadTruffle is a SaaS company that provides texting, web chat, form widgets, and lead-response automations including flows for channels like Yelp, Thumbtack, and Google LSA.
